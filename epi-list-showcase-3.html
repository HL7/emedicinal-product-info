📌 Key FHIR List Fields Used

Field Value(s) Purpose
List.status current Indicates this list is actively maintained
List.mode working A live working list that changes over time
List.code LOINC 34390-5 Classifies list as a Human Pharmacy Label catalogue
List.entry.item Bundle reference Points to the ePI Bundle (the actual document)
List.entry.flag Free text / coded Approval status, template type, or Master vs Translation
List.entry.deleted true / false Marks withdrawn ePIs (requires mode=working)
